# CarrosselAnimado

🌐 Landing Page

Projeto de Landing Page desenvolvido com foco em estruturação responsiva, design moderno e boas práticas de desenvolvimento Front-End.

Funcionalidades

- Layout moderno e responsivo
- Estrutura semântica em HTML
- Estilização com CSS
- Adaptação para diferentes tamanhos de tela
- Organização visual voltada para experiência do usuário

Formatação e linguagem usada

- HTML5
- JavaScript
- CSS3

Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de praticar a criação de páginas institucionais responsivas, aplicando conceitos de:

- Estruturação semântica com HTML
- Estilização com CSS
- Responsividade
- Organização visual de elementos
- Boas práticas de layout

Aprendizados

Durante o desenvolvimento desta Landing Page, foram aprimoradas habilidades em:

- Construção de layouts responsivos
- Posicionamento de elementos com CSS
- Hierarquia visual
- Organização e estilização de componentes
